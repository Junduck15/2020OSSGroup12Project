# 주제
팀원 정보 관리 시스템

# 기능
팀 혹은 반에서 학생들의 정보를 관리하는 CRUD 시스템이다. 기능으로는 학생들의 정보를 읽어들이고 추가하고 수정하고 삭제하고 검색하고 정보를 저장하고 불러들이는 기능들이 존재한다.
